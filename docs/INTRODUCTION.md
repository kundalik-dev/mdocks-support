# Introduction to mDocks

Welcome to **mDocks**! 

**mDocks** is a modern, fast, local-first Markdown reader and browser for your local notes, folders, and GitHub repositories. It provides a clean, clutter-free reading environment that turns flat Markdown files into interactive documentation, with beautiful styling, syntax highlighting, and an automated table of contents.

---

## 🔒 Privacy First (Our Guarantee)

We believe that your personal notes, project documentation, and private repositories should remain strictly yours. 

* **No Server Uploads:** Unlike traditional cloud-based readers, mDocks **never** uploads your Markdown files or folder paths to any external server. 
* **Entirely Local:** Your files are processed, parsed, and rendered entirely on your local machine within the secure sandbox of your web browser.
* **Secure Sandbox Storage:** Any configuration settings, local server paths, and permission keys are stored locally in your browser's IndexedDB storage and never shared.

---

## ⚡ The Three Workspace Modes

To accommodate different reading needs, mDocks operates in three simple modes:

1. 📁 **Browser Files Mode**
   - **What it is:** A zero-installation mode to open local directories and individual files on your computer.
   - **Best for:** Reading local documentation folders, note-taking archives, or single Markdown files.

2. 💻 **Local Server Mode**
   - **What it is:** An integration with a lightweight terminal helper that allows you to clone public or private GitHub repositories and read them completely offline.
   - **Best for:** Teams or developers who want to browse and read official GitHub repositories without an active internet connection.

3. ✨ **Demo Docs Mode**
   - **What it is:** A pre-bundled interactive document to let you experience the full capabilities of the mDocks viewer right away without opening any of your own files.
