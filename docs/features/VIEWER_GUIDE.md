# Viewer Guide & UI Navigation

The **mDocks** viewer is designed for a distraction-free, highly readable documentation experience. This guide will show you how to navigate the reader interface.

---

## 📐 Interface Layout

The viewer is split into three easy-to-use panels:

1. **Left Sidebar:** Displays the **Source Switcher** (for switching between different folders and repositories) and the collapsible **File Tree**.
2. **Center Canvas:** The reader workspace where your Markdown files are formatted and rendered.
3. **Right Outline:** The **Table of Contents** which tracks and displays headings from your active document.

---

## 🔍 Document Outline (Table of Contents)

* **Interactive Outline:** The right panel automatically extracts headings (`H1`, `H2`, `H3`) from your document.
* **Scroll Tracking:** As you read, the outline highlights your active section in real-time.
* **Easy Jumps:** Click on any heading in the outline to instantly scroll to that section.

---

## ⌨️ Keyboard Shortcuts

For rapid navigation without a mouse, mDocks supports standard hotkeys:

* **`s`** - Toggle Left Sidebar (gives you a full-width reading view).
* **`w`** - Toggle Right Outline (Table of Contents).
* **`a`** - Go to the **Previous** file in the active folder list.
* **`d`** - Go to the **Next** file in the active folder list.

---

## 🎨 Themes & Customization

* **Dark & Light Themes:** Toggle between clean light mode and gorgeous dark mode using the theme button (sun/moon icon) in the layout.
* **Persisted Settings:** Your theme preference is remembered and loaded automatically next time you open mDocks.
* **Code Highlights & Copy:** Code snippets are rendered with professional syntax highlighting. Hover over any code box to reveal a quick-copy button.
